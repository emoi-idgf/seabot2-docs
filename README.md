# seabot2-docs

## Seabot2 User Manual Sphinx Documentation

```bash
cd user-manual-docs
```

Build with
```bash
make html
```

Open with
```bash
firefox ./build/html/index.html
```

## seabot2-ros C++ API Doxygen

```bash
cd api-docs
```

Build with
```bash
doxygen
```

Open with
```bash
firefox ./source/_doxygen/html/index.html
```
